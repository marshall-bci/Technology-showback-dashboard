How row-level access works
When a user logs in via BCI SSO, they only see the rows where the GL code and branch code both match their allow-lists.

Admins see all rows.
Viewers must have at least one GL code granted to see any data.
Finance can upload new data but has the same row visibility as their permissions allow.
